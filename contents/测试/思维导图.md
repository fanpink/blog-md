# 项目开发思维导图

## 核心功能
- 实时Markdown渲染
  - 支持CommonMark规范
  - 扩展功能：
    * 表格
    * 任务列表
    * 数学公式
- 文件树自动更新
  - WebSocket通知机制
  - 前端自动刷新

## 技术栈
```mermaid
graph TD
  A[前端] --> B[HTML/CSS]
  A --> C[JavaScript]
  D[后端] --> E[Node.js]
  D --> F[Express]
  G[工具链] --> H[Webpack]
  G --> I[Babel]
```

## 开发计划
1. 第一阶段：基础功能
   - [x] Markdown解析
   - [x] 文件监听
   - [ ] 实时预览

2. 第二阶段：增强功能
   - 数学公式支持
   - 代码高亮
   - 导出PDF
